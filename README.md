# Retail Store Sales Analysis

## Project Overview
This project analyzes over 1,000 retail transactions from online and in-store purchases to derive business insights and recommendations.

## Assignment Sections
- **Section A**: Data Cleaning (Q1-Q3)
- **Section B**: Data Transformation (Q4-Q5)
- **Section C**: Data Analysis (Q6-Q8)
- **Section D**: Visualization (Q9)
- **Section E**: Advanced Analysis - RFM Customer Segmentation (Q10)
- **Section F**: Insight Report (Q11)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Place `retail_store_sales.csv` in `data/raw/`
3. Run `python scripts/run_all_analysis.py`
4. View results in `reports/` folder
